Important notes:
• IR sensor OUT is active LOW — reads LOW when object detected, HIGH when clear.
• Use INPUT_PULLUP or the sensor's built-in pull-up resistor (most FC-51 / E18-D80NK modules have one).
• Do NOT connect Uno TX/RX (D0/D1) to ESP32 while uploading sketches — disconnect first, then reconnect.
• ESP32 is 3.3V logic — use a voltage divider (10kΩ / 20kΩ) on the Uno TX → ESP32 RX line.
• Uno RX can connect directly to ESP32 TX (3.3V → 5V is tolerated by Uno).
